## 第一部分：视觉风格规范

### 1.1 设计定位

**关键词：** 克制 · 温润 · 呼吸感 · 东方现代

整体调性定位为「有温度的现代主义」——避免冷峻的极简或繁复的装饰，以温暖的 neutral 色系、克制的排版节奏、柔和的光影层次，传递一位成熟室内设计师的审美品味。

**参考图分析结论：**
- 5 张参考图整体明度均在 195–219 区间，**整体偏亮，浅色/白色为主**
- 非灰色区以暖灰、棕褐调为主：`#e0d8d8`（暖灰）、`#d0c8c0`（暖米灰）、`#c8b8b0`（褐灰）、`#a08878`（暖铜棕）
- 深色区域集中在深棕 `#301818`、`#281010`，而非纯黑
- 参考图呈现清晰的横向分区（平均 5–8 个区块），背景色交替形成自然节奏

---

### 1.2 配色方案

基于参考图的暖调中性色系，结合 PRD 中「室内设计师」的身份属性，建立以下色板：

| 色阶 | 色值 | 色相描述 | 用途 |
|---|---|---|---|
| **主背景** | `#faf8f5` | 暖白（微黄底） | 页面主背景 |
| **次背景** | `#ffffff` | 纯白 | 卡片 / 区块交替背景 |
| **深底** | `#2c241e` | 深棕 | CTA 区域深色背景 |
| **最深底** | `#1e1814` | 最深棕 | Footer |
| **主文字** | `#1a1a1a` | 近黑 | 标题、重要文字 |
| **次文字** | `#3d3229` | 暖深棕 | 正文段落 |
| **辅助文字** | `#4a3f36` | 中深棕 | 描述性文字 |
| **弱化文字** | `#8c7b6e` | 暖灰棕 | 元信息（时间、地点标签） |
| **更弱文字** | `#6b5e53` | 中暖棕 | 次级元信息 |
| **强调色** | `#b8977e` | 暖铜色 | 链接、标签、装饰元素 |
| **强调深** | `#8c6b55` | 深铜色 | hover 态 |
| **荣誉标签背景** | `#fdf2e9` | 暖杏色 | 奖项标签底色 |
| **荣誉标签文字** | `#a0522d` | 棕红 | 奖项标签文字 |
| **分隔线** | `#d4c5b9` | 淡暖灰 | 分割线 / 时间线轴 |

**色板推导逻辑：**
- 主背景 `#faf8f5`：参考图中大面积高光区域并非冷调的 `#fff`，而是带微黄底色的暖白，较纯白多了一层温度感
- 深色区域 `#2c241e` / `#1e1814`：参考图中深色块并非 `#000`，而是偏棕的深色 —— 对应室内设计中「木饰面深色」的意象
- 强调色 `#b8977e`（暖铜）：源自参考图中暖色调集中区域 `#a08878` ~ `#c8b8b0` 的中值，类似铜质五金件的色泽，含蓄而有质感
- 文字色系从 `#1a1a1a` → `#3d3229` → `#8c7b6e` 形成暖调渐变，避免纯黑到纯灰的生硬

**配比原则：**
- 70% 暖白 / 白色背景
- 20% 文字色系（黑 → 深棕渐变）
- 8% 强调色（暖铜 `#b8977e`）
- 2% 装饰 / 分隔线

### 1.3 字体规范

参考图中文字排版以清晰的无衬线体为主，标题部分有衬线体的装饰性使用。结合「室内设计师」身份，采用**衬线点睛 + 无衬线为体**的策略：

| 层级 | 字体 | 大小 | 字重 | 用途 |
|---|---|---|---|---|
| **H0** | Noto Serif SC | 72px | 700 | Hero 姓名 |
| **H1** | 系统无衬线 | 36px | 600 | 板块标题 |
| **H2** | 系统无衬线 | 28px | 600 | 项目名称 |
| **H3** | 系统无衬线 | 18px | 600 | 时间线角色 |
| **Body L** | 系统无衬线 | 17px | 400 | 关于我正文 |
| **Body M** | 系统无衬线 | 16px | 400 | 项目描述 |
| **Body S** | 系统无衬线 | 14–15px | 400 | 标签 / 联系信息 |
| **Caption** | 系统无衬线 | 13px | 500 | 板块标签（ABOUT / PROJECTS） |
| **Number** | Noto Serif SC | 48px | 700 | 亮点数字 |

**字体族谱：**

```css
/* 衬线 — 仅用于 Hero 姓名 & 亮点数字 */
font-family: 'Noto Serif SC', 'SimSun', 'STSong', serif;

/* 无衬线 — 正文全局 */
font-family: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei',
             'Noto Sans SC', sans-serif;
```

**排版节奏：**
- 板块间距：100px（桌面）/ 64px（移动）
- 标题与内容间距：48px
- 段落间距：20px
- 行高：1.8（中文正文）
- 字间距：标题 +1~3px letter-spacing，正文默认

### 1.4 圆角规范

参考图呈现的是建筑/室内设计风格，圆角应克制，保持「建筑感」——避免全局大圆角。

| 组件 | 圆角 | 说明 |
|---|---|---|
| **项目图片** | `4px` | 微圆角，保持建筑感 |
| **技能标签** | `30px` | 全圆角胶囊形 |
| **证书卡片** | `8px` | 柔和圆角 |
| **CTA卡片** | `8px` | 与证书卡片统一 |
| **荣誉标签** | `20px` | 小胶囊 |
| **按钮 / 链接下划线** | 无圆角，用 `border-bottom` 实现 |

**原则：** 功能性组件（图片）保持锐利克制 → 4px；交互性组件（标签、卡片）适度圆润 → 8–30px。避免全局大圆角，保持专业感。

---

### 1.5 阴影规范

参照参考图中光影层次，阴影采用多层叠加策略，全部使用黑色透明度控制，无彩色阴影。

| 层级 | 阴影值 | 使用场景 |
|---|---|---|
| **Level 1** | `0 2px 12px rgba(0,0,0,0.04)` | 技能标签、证书卡片（静态） |
| **Level 2** | `0 6px 20px rgba(0,0,0,0.08)` | 技能标签（hover 浮起） |
| **Level 3** | `0 20px 60px rgba(0,0,0,0.08)` | 项目图片 |
| **Level 4** | `0 8px 40px rgba(0,0,0,0.12)` | Hero 头像 |

**原则：** 阴影只用黑色，通过透明度和模糊半径区分层级。不使用彩色阴影。阴影数量控制在 4 级以内，避免视觉噪音。

### 1.6 卡片样式

**基础卡片 — 技能标签胶囊**
```
白底 #ffffff + Level 1 阴影 + 30px 圆角胶囊
内部 flex：emoji 图标 (18px) + 文字 (14px)，gap: 8px
padding: 10px 22px
hover → translateY(-2px) + Level 2 阴影 + 0.3s ease
```

**证书卡片**
```
白底 #ffffff + Level 1 阴影 + 8px 圆角
flex 横向：图标 (20px) + 文字 (14px)，gap: 12px
padding: 16px 20px
静态，无 hover 动效
```

**CTA 卡片**
```
深色半透明底 rgba(255,255,255,0.04) + 8px 圆角
flex 纵向居中：图标 (32px) → 标签 → 值 → 提示
padding: 32px 20px
hover → background: rgba(255,255,255,0.08) + 0.3s
```

**项目图片卡片**
```
aspect-ratio: 4/3，overflow: hidden，border-radius: 4px
Level 3 阴影
hover → img scale(1.03)，transition: 0.6s ease
```

### 1.7 整体协调性

```
┌────────────────────────────────────────────┐
│                 暖白 #faf8f5               │  ← Hero
│         姓名(衬线) · 头像 · 联系方式        │
│         ↓ 渐变分割线（铜色→透明）            │
├────────────────────────────────────────────┤
│                 纯白 #ffffff               │  ← About
│         两栏 Grid：文字 + 亮点数字           │
├────────────────────────────────────────────┤
│                 暖白 #faf8f5               │  ← Projects
│         图片左/右交替 + 描述                │
├────────────────────────────────────────────┤
│                 纯白 #ffffff               │  ← Experience
│         竖线时间线                          │
├────────────────────────────────────────────┤
│                 暖白 #faf8f5               │  ← Skills
│         胶囊标签网格 + 证书卡片             │
├────────────────────────────────────────────┤
│                 深棕 #2c241e               │  ← CTA
│         三栏联系方式                        │
├────────────────────────────────────────────┤
│                 最深 #1e1814               │  ← Footer
└────────────────────────────────────────────┘
```

**协调法则：**
1. **背景交替：** 暖白 ↔ 纯白交替，形成自然视觉分段，无需硬分割线
2. **留白呼吸：** 每个板块 100px 上下间距，内容区最大宽度 1080px
3. **色彩克制：** 仅一个强调色 `#b8977e`，贯穿 label / 时间点 / 链接 / 数字
4. **字体层次：** 衬线仅用于 Hero 和数字亮点，其余全无衬线
5. **动效统一：** 所有过渡 `0.3s ease`，仅位移和透明度变化，无弹簧 / 弹跳

## 第二部分：六大板块 UI 规格

### 板块一：Hero（首屏）

**目标：** 1 秒内建立身份认知 —— 我是谁、做什么、怎么联系。

#### 布局

```
         ┌─────────────────────────┐
         │                         │
         │     [圆形头像 140px]     │
         │     object-fit: cover   │
         │                         │
         │     印  昱  橙          │  ← 72px Noto Serif SC
         │     letter-spacing:8px  │     字重 700
         │                         │
         │  12 年全流程室内设计师    │  ← 18px 无衬线
         │  从概念到落地，创造兼具   │     color: #8c7b6e
         │  功能与美学的现代居所     │
         │                         │
         │  📞 电话  ✉️ 邮箱  📍 地点│  ← 15px flex 三列
         │                         │
         └─────────────────────────┘
              ↓ 底部渐变分割线
        linear-gradient(to right,
          transparent, #d4c5b9, transparent)
```

#### 组件清单

| 组件 | HTML 元素 | 关键 CSS |
|---|---|---|
| 头像 | `<img class="hero-photo">` | `w140 h140 rounded-full object-cover` + Level 4 阴影 |
| 姓名 | `<h1 class="hero-name">` | 衬线体, 72px/700, `letter-spacing: 8px`, `color: #1a1a1a` |
| 定位语 | `<p class="hero-tagline">` | 18px/400, `color: #8c7b6e`, `letter-spacing: 2px`, 两行 |
| 联系方式 | `<div class="hero-contact">` | flex 三列等距, `gap: 40px`, 15px, `color: #6b5e53` |
| 容器 | `<section class="hero">` | `min-height: 100vh`, flex 居中, `background: #faf8f5` |

#### 交互效果

- **头像/姓名/文字：** 无 hover 动效，纯静态展示
- **页面加载：** 可选整体淡入 `opacity 0→1`, `duration: 0.8s`, `ease`
- **底部渐变分割线：** `::after` 伪元素，绝对定位，`height: 1px`, `linear-gradient`
- **联系方式各项：** 无点击交互（纯信息展示）

#### 响应式 (≤768px)

- 姓名 → 40px, `letter-spacing: 4px`
- 头像 → 100px
- 联系方式 → 纵向堆叠，`flex-direction: column`, `gap: 12px`
- Hero 最小高度适当降低

| 荣誉标签 | — | 静态，无交互 | — |

#### 响应式 (≤768px)

- Grid → 单栏，图片始终在上（`.reverse` 的图片 order 重置为 0）
- `gap: 28px`
- 项目名 22px
- `margin-bottom: 64px`

### C. 参考图对照表

| 参考图 | 整体明度 | 主导色调 | 区块模式 | 对本设计的启示 |
|---|---|---|---|---|
| 参考图1 | 219.5（最亮） | 暖灰 + 深棕文字 | 6 区交替 | 整体明亮、暖白基底 |
| 参考图2 | 197.0 | 暖灰 + 褐色点缀 | 6 区交替 | 中灰层次过渡自然 |
| 参考图3 | 208.5 | 暖灰 + 铜棕 | 8 区细分区 | 多层次灰度语言 |
| 参考图4 | 195.6 | 纯灰白 | 4 区简洁 | 极简克制、留白为主 |
| 参考图5 | 208.9 | 纯灰白 | 10 区精细 | 段落节奏感强 |

**共同特征：**
- 全部以亮色/白色为主背景（明度 195–220）
- 深色区域使用偏棕色而非纯黑
- 横向分区形成自然视觉节奏
- 强调色使用暖铜/褐调，面积克制
- 留白充足，信息密度低
